## How to Customize Your Landing Page

You can easily change the content by editing the `index.html` file in Cursor:

### Change the Main Title
Look for this line and replace the text:
```html
<h1>Welcome to Your Amazing Landing Page</h1>
```

### Change Colors
The gradient background uses these colors:
- Purple: `#667eea`
- Dark Purple: `#764ba2`

Find these in the CSS section and replace with any color codes you like!

### Change the Button Text
Find this line:
```html
<a href="#features" class="cta-button">Learn More</a>
```

### Add Your Name
At the bottom, replace "Your Name":
```html
<p>&copy; 2026 Your Name. Built with Cursor and enthusiasm! ðŸŽ‰</p>
```
